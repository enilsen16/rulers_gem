# Rulers

A rack based gem similar to Rails.

## Installation

Add this line to your application's Gemfile:

    gem 'rulers', :git => 'git://github.com/enilsen16/rulers.git'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rulers

## Usage

Right now you can start a new server with:

      rackup -p 3001

Then navigate to:

      http://localhost:3001
